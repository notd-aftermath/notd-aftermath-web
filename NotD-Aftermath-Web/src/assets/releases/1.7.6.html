General Theme of Change

  • Gameplay Changes
  • ALICE Changes
  • Bug Fixes

This is a very small release, focused on fixing a few major bugs.

Changes

Gameplay Changes

  • The chapter transition cutscene no longer begins instantly after Eris is killed in Revelation.

ALICE Changes

  • Added sitrep 4 to display experience gained in the current game.
  • Added short forms for a few commands (e.g. -c for clear or -t for team)

Bug Fixes

  • Fixed flare vision not being shared with allies during Lost Cause.
  • Fixed issue with game progression getting stuck when closing last cave at certain points in the game.
  • Fixed text being cut in the quest menu.
  • Fixed item restrictions not being defined properly (pred can now use M-29 again)
  • Specimen 13-A can no longer be ctrl + clicked.
  • To simplify trigger code and resolve XP issues, players still in game and leavers will now be considered the same for XP gain.
  • Players that leave before picking a class are no longer counted as a requirement for service medals.
  • Save files of the red player are no longer created while watching a replay.
  • Using hold position from an unpathable location no longer teleports the unit.
  • Killing bunnies in the ending cinematic no longer gives experience to all players, only to the one clicking them.
  • Added fix to prevent the sprint ability from lasting forever sometimes.
  • [Flamethrower] Fixed fuel transmission tooltip not matching attack range and lvl 5 not working.
