General Theme of Change

  • ALICE Changes
  • Visual changes
  • Bug Fixes
  • Internal changes

This version contains a few gameplay changes, some quality of life improvements and the usual bugfixes.

Changes

Gameplay Changes

  • Reforged graphics mode is now disabled. The map will always load in classic graphics regardless of player options.
  • Updated items system to allow having disabled items in the inventory (e.g. Blastocyst on MQ-3 Predator or Arclite M23 Hecate on classes other than Flamethrower)
  • Civilians can no longer use primary weapons (but they can still carry them). They can still use light weapons (M320, Benelli RMS-8 and Browning RMS-12A)
  • Civilians can no longer pickup heavy explosives (SMAW, XML-9C Shiva and FGM-148 Javelin Missile)
  • The FGM-148 Javelin Missile can no longer be carried at the same time as other heavy weapons.
  • Enabled native WC3 item stacking introduced in Reforged. This only applies to items that stack with themselves, so for example M320 + HEDP is excluded.
  • Updated containment titan nuke ping timers to 120/60/30 seconds based on difficulty.
  • Reduced number of titan nuke drops from 5 to 4 on hell difficulty.
  • Osiris can no longer spawn during USS Independence escort mission.
  • Etaphreven can no longer spawn in Magnificent Bastards.
  • Players are now invulnerable during chapter transition.
  • The number of EM Sphere release codes required is now reduced when a player leaves the game.
  • If all players die while EM Sphere is charging, or the player with the last required activation code dies/leaves, the EM Sphere will now be activated.
  • All spawns from caves and waterfalls are now random (with the exception of the 1st wave at the start of the game and the spawns for the USS Independence event)
  • Enemies will no longer spawn from closed caves (total amount is not reduced, they just spawn elsewhere)
  • Golf marines no longer spawn in caves or waterfalls in Lost Cause.
  • Havok can now see invisible units in hell difficulty.
  • Slightly reduced the damage of danger zones from FM and Satchels.
  • Multiple players can now use laser designator on the same target and they will all have vision of it.
  • Enabled safe location system for all random enemy spawns to prevent them from happening within player range.
  • Forward stations are no longer owned by players in Deathmatch mode.
  • Players can no longer spawn inside the tunnel.
  • The sheep pen fences are now destructible.
  • The EM Sphere release codes are now randomly generated with the pattern [A-Z1-9]{4} instead of chosen from a random predefined set.
  • Parasite infections are now properly counted as ailments.
  • The timeout to switch to Revelation if Lyganesh is not attacked in Delta is now based on the distance it spawns from players.
  • Removed time-based seed used for the -random command.
  • Killing Prometheus after Residence 41 has been destroyed no longer gives XP.
  • [Forward Observer]: Reduced cost of White Phosphorus and Illumination Rounds from 5 shells to 3 shells.
  • [Forward Observer]: Removed Ordnance Handling ability.
  • [Combat Engineer]: Failing to place the Communications Array due to location will now reset the ability cooldown.
  • [Assaultman]: Increased energy cost of claymores from 25 to 35.
  • [Assaultman]: Added Ordnance Handling ability.
  • [MQ-3 Predator]: Added searchlight ability.

ALICE Changes

  • Added discord link on all end game messages and quest menu (website link is still there, we now have both).
  • Each player can now have a different sitrep mode.
  • Removed command to set mission objective using sitrep.
  • Context tagging now specifies the medical condition of the target (e.g. "I have open wounds" instead of "I need medical assistance").
  • Context tagging now allows targetting allied explosives.
  • Added total ailments to -ac.
  • Added necrotroph to -ac.
  • Removed -aca message at the start of the game. The functionality is still available but won't be advertised to avoid confusing new players.
  • Added tutorial message explaining poppers to new players.

Visual Changes

  • Mundus no longer has a health bar.
  • Updated model for MQ-3 Predator (model by ILH)
  • Updated model for the sheep pen fences (model by Grey Knight)
  • Camera view modes and first person modes are no longer enabled while watching a replay.

Bug Fixes

  • Fixed tooltip for Arclite M23 Hecate to reflect that increased attack speed amount is 20%, not 30%.
  • Fixed tooltip for SAM-R to reflect that increased sight range is 300, not 250.
  • Added extra check to the AI smoke grenades trigger to prevent it from running on player controlled units.
  • Fixed bug that allowed saving 4 times per game instead of 3.
  • Fixed incorrect initialization for containment spawns trigger. This may result in an increase of up to 80% for some spawns.
  • Ailment reductions from abilities are no longer player-wide. Meaning marine reinforcements or rescued civilians no longer get these benefits.
  • Ailment reduction from antibiotics now only works when applied to heroes.
  • Fix Machine Gunner, Combat Engineer and Designated Marksman not allowed for privates in Bootcamp mode.
  • Spore parasite spawn timer now properly scales to the number of players.
  • Players that leave the game before selecting a class are no longer considered alive for balancing/spawns.
  • Hero selection units for empty slots are now removed after map start.
  • Player handicap no longer has any effect on the game.
  • Removed permanent camera shake in Deathmatch mode.
  • Fixed Assaultman upgrades not working after reviving in Deathmatch mode.
  • Fixed some difficulty modifiers (higher immortal respawn rate, ailment chance) not being applied with high SR.
  • Refactored movement speed adjustment system to prevent unintended speed changes.
  • Fixed Magnificent Bastards LZs being generated after the "no further landings" message.
  • Fixed sprinting while having a fracture not dealing damage.
  • Fixed delta helicopter cleanup impacting player controlled helicopters.
  • The 5 minutes Moirae timer no longer breaks after 1h.
  • Fix Ogilvy not changing position properly after he gets his gun.
  • [Rifleman]: Critical Strike is now leveled correctly.
  • [Flamethrower]: Fixed incorrect hotkey for learning Fortitudine.
  • [Combat Engineer]: Failing to place the Communications Array due to location no longer gives back more energy than the cost of the ability.

Internal Changes (not interesting for players)

  • Added basic W3MMD integration.
  • Moved all trigger code to a Visual Studio Code workspace outside the map to facilitate development and improve version control.
  • Refactored all player data into a single struct.
  • Moved constant values to a separate file.
