Another small bugfix release. Also includes a small test for Traditional Chinese localization.

Changes

Gameplay Changes

  • [Combat Engineer] The armor reduction effect now lasts the same as the duration of the net.
  • [Forward Observer] Increased white phosphorus armor reduction cast duration by 1s.

ALICE Changes

  • Moved ALICE command list from quest menu to main menu (F10 > H)

Bug Fixes

  • Fix experience bonus on MB win.
  • EM Sphere release no longer counts for hits taken.
  • Hits after the game is completed no longer count for hits taken.
  • Fixed infinite javelin missile exploit.
  • Fixed infinite Order 66 nukes bug.
  • [Combat Engineer] Fixed Nanosteel Net tooltip showing incorrect duration.

Other

  • Night Vision is no longer visible in replays.
  • Added base localization files for Traditional Chinese as a test for multi-language release. Very few things are translated so far, but it will be worked on over time.
