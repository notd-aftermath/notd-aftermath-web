The new save system update is finally here, FMar is a thing of the past!

Changes

Gameplay Changes

  • Added new save system with maximum of 100M experience.
  • Saves are now tied to full Battle.net BattleTags including numbers (e.g. Player#1234)
  • Added First Sergeant and Master Gunnery Sergeant ranks right after Gunnery Sergeant. All ranks after this have been pushed by 1, with max rank now being General at 2M experience. The Field Marshal rank has been removed. This only impacts the officer abilities that get unlocked at 2Lt. Class requirements have not changed.
  • Added falling shell sound to Lieutenant Oliver Fire Missions.
  • Monofilament cartridge and MD5 mines no longer make targets hit briefly invulnerable.
  • [Assaultman] The Javelin CLU can now only be used by the Assaultman. Reduced considerably the weight of the item (total is now 12 missile + 4 CLU)
  • [Forward Observer] The earshock area of Orbital Artillery now goes up as the AoE of the ability increases instead of being the maximum range from the first level.
  • [Combat Engineer] Added fix to prevent the rapid fire ability from lasting forever sometimes.
  • Changed the -tl command to ping both Alpha and Bravo squads
  • Added short form of the -prime command (-p) for adjusting the satchel primer.
  • The Ctrl + click tag system can now be used after you die.
  • Players leaving before selecting a class no longer count as a casualty in terms of marine reinforcements.
  • Pet bite can no longer be casted on the pet.
  • Fixed game getting stuck if the USS Independence dies right after it spawns.
  • Experience is no longer subtracted when there is a negative XP gain factor.
  • The SMAW area for damaging items and trees now scales up with SMAW level, same as the unit damage area.
  • Artillery shells can no longer go into negative values.
  • Slight adjustments to position of some terrain doodads (mainly trees).
  • Changed -view and -fps interaction, the game now returns you to your previously selected -view after leaving fps mode

Visual Changes

  • Removed highlighted hotkeys from all ability tooltips to simplify localization.
  • [Forward Observer] Updated Orbital Artillery effect to use projectiles instead of an explosion. Description of the ability has been updated to reflect this (kinetic projectiles fired from orbit instead of missiles).
  • [Forward Observer] Added passive ability icon for Perimeter Intelligence.
  • [Force Recon] Added passive ability icon for Raven Surveillance.
  • Changed the loading screen back to the classical one
  • Hero attributes are now hidden.
  • Fixed a few typos in the mission script.
  • Fixed unit descriptions for AH-1W Cobra and AH-64D Apache Longbow.

Bug Fixes

  • [Forward Observer] Orbital artillery can no longer target bodies (should fix a visual bug that occurred sometimes)
  • Fixed Osiris marine reaction messages not working.
  • Reverted the change that disabled NVG in replays to prevent bugged replays (camera changes are still disabled in replays, those should be fine)
  • Added error correcting code to prevent bugged saves that changed CR or MOS.

Other

  • Autoload now uses the sync natives. Removed the Sync, SyncInteger and PlayerUtils vJass libraries since they are no longer required.
  • Optimized a bit the ctrl + click tag system (removed one network-synced call).
